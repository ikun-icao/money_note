# 理财收益记录

一个精美的个人理财收益追踪网页应用，采用杂志风格设计，帮助您记录和分析每日理财收益。

## 功能特点

### 📊 核心功能
- **每日收益记录**：轻松记录每日理财收益金额和收益率
- **收益曲线图**：直观展示累计总盈亏百分比的变化趋势
- **收益热力图**：GitHub风格的热力图，按年份展示每日收益情况
- **理财笔记**：记录理财心得和投资策略

### 🎨 视觉特效
- **负收益特效**：
  - 雪花飘落动态背景
  - 数字冰块发光特效
  - 绿色系热力图标记

- **正收益特效**：
  - 花瓣飘落动态背景
  - 数字火焰燃烧特效
  - 红色系热力图标记

### 🎯 设计特点
- 杂志风格UI设计，不对称网格布局
- 强层次结构和大胆排版
- 白底渐红页眉设计
- 响应式布局，适配各种设备

## 技术栈

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **图表库**：Chart.js（用于收益曲线图）
- **数据存储**：localStorage（本地持久化存储）
- **动画效果**：CSS3动画 + JavaScript动态生成
